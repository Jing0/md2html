# MD2HTML

a C implementation of the Markdown to HTML system.

## Supported

* `Customized CSS`
* `Headers` ( from # to #### )
* `Part Separator`( --- ).
* `Link` ( (name)[url] )
* `Blockquote` ( > )
* `Image` ( !(name)[url] )
